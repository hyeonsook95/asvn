# ASVN

## 프로젝트 개요
2016년, 시스템 프로그래밍 학기 과제용으로 만든 팀 프로젝트입니다.
FTP 프로그래밍을 이용하여 서버와 클라이언트 간의 파일 전송을 구현하고, 버전관리를 위한 기능을 구현합니다.

## 구현된 기능

### 로그 기능
* 로그를 기록하는 기능
* 기록된 로그를 확인하는 기능

### 네트워크 기능
* 클라이언트 측에서 서버에게 처리를 요청하는 기능
* 서버 측에서 클라이언트에게 처리를 요청하는 기능
